# A Chat app
This is a chat app where you can type and recieve and send messages.


![Application running](docImages/Header-image.png)

## Getting started

* `npm install`
* `npm start` or if you have nodemon locally `nodemon`


## What's Used

* `ejs: 2.5.8,`
* `express: 4.16.3,`
* `nodemon: 1.17.3,`
* `save: 2.3.2`
* `socket.io: 2.1.0`
